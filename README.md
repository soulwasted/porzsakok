# porzsakok

Use node.js v6.

### Try nvm
```
$ nvm i 6
$ nvm use 6
$ node -v // should be node version 6.x.x
```


## Set Environment

### Repo
```
$ gh repo clone soulwasted/porzsakok
```

### Install
```
$ npm i
$ bower i
# $ npm install -g browser-sync
```

### Run
```
$ gulp
# $ browser-sync start --server www
```

### Iconfont
```
$ gulp iconfont
```

## Production
```
$ gulp production
```

Good luck ;)
